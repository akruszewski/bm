# bm - manage bookmars from cli #

```
NAME:
   bm - manage bookmars from cli

USAGE:
   bm [global options] command [command options] [arguments...]

COMMANDS:
   init     init bm repository
   add      add bookmark to repository
   update   update bookmark
   list     list bookmarks
   get      get bookmark
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --path value  path to bm database (default: "~/.bm/bm.csv")
   --help, -h    show help (default: false)
```
